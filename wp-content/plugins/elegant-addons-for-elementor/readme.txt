== Installation ==

= Option 1. =

- Upload the entire `elegant-addons-for-elementor` folder to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2. =

- From your wp-admin —> Plugins —> Add New —> Search "Elegant Addons for Elementor" —> Click install.
- After plugin is downloaded, click active plugin

= After the plugin active =

- You will find 'Elegant Addons' Widget category under your elementor frontend page builder panel.

= Note =

- You must have 'Elementor page builder' plugin installed and activated to able to use this plugin.
